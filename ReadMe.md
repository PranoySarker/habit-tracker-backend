# Habit Tracker API

Backend for the Habit Tracker app.
Built with: Node.js, Express, TypeScript, Prisma, PostgreSQL

## Setup

1. Clone the repo
2. Run `npm install`
3. Copy `.env.example` and fill in your values
4. Run `npm run dev`

## Scripts

- `npm run dev` — start development server
- `npm run build` — compile TypeScript
- `npm start` — run compiled app
